# Mintailor

## About

Mintailor is a Vue Component Library which was built on TailwindCSS.

## Usage

```bash
npm i -D mintailor
```

**tailwind.config.js**

```js
export default {
  content: ['./node_modules/mintailor/src/components/**/*.{vue,js,ts,jsx,tsx}']
};
```

```js
import { Mintailor } from 'mintailor';
```

```xml
<Mintailor>
  <!-- your application -->
</Mintailor>
```
