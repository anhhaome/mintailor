# Mintailor

## About

Mintailor is a Vue Component Library which was built on TailwindCSS.

## Usage

```bash
npm i -D mintailor
```

**tailwind.config.js**

```js
import { createConfig } from 'mintailor/config';

/** @type {import('tailwindcss').Config} */
export default createConfig({
  /* current config */
});
```

```js
import { Mintailor } from 'mintailor';
```

```xml
<Mintailor>
  <!-- your application -->
</Mintailor>
```
